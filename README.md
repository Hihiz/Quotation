# Quotation
API сервис с аутентификацией JWT token, для просмотра курса валют банка.

## Стек
* ASP.NET Core 8.0
* JWT token
* MSSQL
* EF Core
* React + TypeScript - client

## Запуск приложения клиента
* Перейти в папку ```quotation.client```
* Установить зависимости
```
npm install
```

* Запуск
```
npm start
```
